# Leaderboard History Data

This folder contains the generated history file for the Europe leaderboard.

## Files

- `history-europe.json` - Europe region leaderboard history (compact columnar format)

## Format

The history file uses a compact columnar format:

```json
{
  "region": "europe",
  "dates": ["2026-01-01", "2026-01-02", ...],
  "players": {
    "PlayerName|de": {
      "n": "PlayerName",
      "c": "de",
      "r": [1, 2, null, 3, ...],
      "th": [["OG", 12345, 0], ...]
    }
  },
  "aliases": { ... },
  "meta": { ... }
}
```

- `r` = rank at each date index (null = absent that day)
- `th` = team history: `[team_tag, team_id, startDateIndex]`

## Generation

This file is auto-generated by `scripts/extract-history.js` during the build process.
It reads daily snapshot files from `data/snapshots/europe/`.

**Do not edit this file manually** - it will be overwritten on each build.
